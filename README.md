## 🚀 CARA INSTALL (GAMPANG BANGET)

### Yang Dibutuhin
```bash
✅ PHP 8.0 atau lebih tinggi
✅ MySQL 5.7 atau lebih tinggi
✅ Apache/Nginx Web Server
✅ XAMPP/Laragon (Recommended buat lokal)
✅ Browser modern (Chrome, Firefox, Edge)
```

### Langkah Install

#### 1️⃣ Download/Clone Project
```bash
git clone https://github.com/VioNetDevelopments/vexalyn-agency.git
cd vexalyn-agency
```

Atau download aja zip-nya langsung dari GitHub, extract, beres.

#### 2️⃣ Setup Database
```bash
# Buka phpMyAdmin (biasanya http://localhost/phpmyadmin)
# Buat database baru
CREATE DATABASE vexalyn_agency;

# Import file SQL
mysql -u root -p vexalyn_agency < db_vexalyn.sql
mysql -u root -p vexalyn_agency < db_vexalyn_updates.sql
```

Atau lewat phpMyAdmin aja, klik Import, pilih file SQL-nya.

#### 3️⃣ Koneksi Database
```php
// config/database.php
$host = "localhost";
$user = "root";
$pass = "";  // Password MySQL lo (default XAMPP kosong)
$db   = "vexalyn_agency";
```

#### 4️⃣ Set Permissions (Kalo di Linux/Mac)
```bash
chmod -R 755 /path/to/vexalyn-agency
chmod -R 777 /path/to/vexalyn-agency/uploads
```

**Kalo Windows?** Gausah pusing, langsung jalan aja biasanya.

#### 5️⃣ Buka di Browser
```
http://localhost/vexalyn-agency/
```

## 🐛 KALO ADA MASALAH

### Masalah Umum

| Masalah | Solusi |
|---------|--------|
| **Database Gagal Koneksi** | Cek `config/database.php` |
| **Session Error** | Pastikan `session_start()` dipanggil |
| **Gambar Gak Muncul** | Cek permission folder assets |
| **Globe Gak Jalan** | Butuh internet (CDN) |
| **Login Gagal** | Cek hash password di database |
| **404 Error** | Cek .htaccess dan URL rewriting |

### Debug Mode
```php
// Aktifin error reporting (Kalo lagi development aja)
ini_set('display_errors', 1);
ini_set('display_startup_errors', 1);
error_reporting(E_ALL);
```
